Properties:
Name Type Description
element Element The dom element for this column.
index Integer The numerical index for this column, the left-most column is zero and the index increases as more columns are added.
ROWS Array An array of Row Objects representing the content inside of this column
V_DRAGS Array An array of Vertical Drag dom elements, these elements have a drag watcher attached to them and a callback function resize two rows in this column.
onDragEnd function A callback function that is executed whenever this column has been drug, and the mouse is released
onVDrag function A callback function that is executed whenever this column is in the process of being drug.
createVDrag function A function to add a Vertical Drag to this column, the vertical drag will enable rows to be resized dynamically.
callbacks Array An array of callback functions to be executed when this column is resized.
executeElementsCallback function This is the function which is executed whenever this column is resized, it then in part executes all of the functions defined in the callback array.
registerCallback function This function is called to register interest in a specific row. A callback function is also passed, this callback will be executed when that specific element is resized.
